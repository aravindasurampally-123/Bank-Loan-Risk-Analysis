# Bank Loan Risk Analysis

## Project Overview
This project analyzes bank loan customer data to identify factors affecting loan default. It includes data preprocessing, machine learning, SQL analysis, and an interactive Power BI dashboard.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- PostgreSQL
- Power BI

## Machine Learning Models
- Logistic Regression
- Random Forest

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Scaling
4. Model Training
5. Model Evaluation
6. Feature Importance
7. SQL Analysis
8. Power BI Dashboard

## Results
- Logistic Regression Accuracy: **96.95%**
- Random Forest Accuracy: **96.45%**

Feature Importance:
- Bank Balance
- Annual Salary
- Employed
